# Projet Tables et ensembles – Java

## Présentation
Ce projet est une adaptation personnelle d’un TP universitaire réalisé en Java dans le cadre du cours de Programmation Avancée (Licence 3 – ISTIC, Université Rennes 1).

L’objectif du projet est d’implémenter et manipuler des structures de données de type tables et ensembles, en respectant des contraintes de conception et de robustesse.

Ce dépôt a été nettoyé, documenté et restructuré dans un objectif de présentation professionnelle (portfolio / recherche de stage).

---

## Compétences mises en œuvre
- Programmation orientée objet en Java
- Implémentation de structures de données
- Manipulation de tables et d'ensembles
- Organisation d’un projet Java
- Lecture et compréhension d’un code existant

---

## Fonctionnalités principales
- Création et manipulation de tables et d'ensembles
- Ajout, suppression et parcours d’éléments

---

## Exécution
Le projet peut être exécuté en lançant la classe `App.java` située dans `src/main/java/fr/istic/pra/l3map/`.
